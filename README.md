## 💻 本地开发
- npm: 10
- node: 21

```
npm install -g yarn
yarn
yarn expo start --tunnel
```
ios端下载 Expo Go App, 连接vpn或代理服务后用ios自带相机扫描终端二维码，即可在 Expo Go 中启动项目
